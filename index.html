truncate-text

This is a web component that will help you to truncate text. You can truncate passing the number of characters or words.

This component was inspired on the AngularJs module angular-truncate

Download on Github

Truncate on 22 words

    
    <p>
        <truncate-text words="22">
            Lorem ipsum dolor sit amet, consectetur adipisicing
            elit. Consequuntur, molestias, consequatur. Ipsam
            adipisci iure, labore facilis sequi natus, nihil
            error explicabo ea pariatur rerum unde possimus
            dolor voluptates, quae ut!
        </truncate-text>
    </p>

Will render:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Consequuntur, molestias, consequatur. Ipsam adipisci iure, labore facilis sequi natus, nihil error explicabo ea pariatur rerum unde possimus dolor voluptates, quae ut!

Truncate on 75 characters. Will not break the last word

            labore facilis sequi natus, nihil error explicabo ea pariatur
            rerum unde possimus dolor voluptates, quae ut!
